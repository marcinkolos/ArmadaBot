# ArmadaBot
 
<b>Old, discontinued</b> project of automatic playing browser game called Armada Battle. Based on Chromium Embedded Framework.

Features:
<ul>
 <li>Setting priorities</li>
 <li>Setting logic</li>
 <li>Works in tray</li>
 <li>Autologin</li>
 <li>Autorepair</li>
 <li>Attacking all kind of enemies</li>
 </ul>
